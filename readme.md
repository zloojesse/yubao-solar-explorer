# 魚寶的 3D 太陽系小探險

一個給魚寶玩的 3D 太陽系互動網站：點選星球後鏡頭會飛近，顯示繁體中文星球小知識，並可挑戰互動太空小問答。

## 特色

- 真 3D 太陽系場景，基於 Three.js / Vite
- 太陽 Bloom 發光、行星貼圖、地球日夜 shader
- 星球 hover outline、點選飛近
- 繁體中文星球資訊卡
- 手機版 responsive HUD
- 保留小行星帶、柯伊伯帶與冥王星，並新增穀神星、妊神星、鳥神星、鬩神星與哈雷彗星
- 互動小知識卡：依照目前選取的星球切換問題與答案

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```
