# 魚寶的 3D 太陽系小探險

一個給魚寶玩的 3D 太陽系互動網站：點選星球後鏡頭會飛近，顯示繁體中文星球小知識，並把「數位化 → 雲端化 → 自動化 → AI 化」與 AI 工作分身養成概念藏進探索旅程。

## 特色

- 真 3D 太陽系場景，基於 Three.js / Vite
- 太陽 Bloom 發光、行星貼圖、地球日夜 shader
- 星球 hover outline、點選飛近
- 繁體中文星球資訊卡
- 手機版 responsive HUD
- 保留小行星帶、柯伊伯帶與冥王星
- 結合前一版課程概念：數位化、雲端化、自動化、AI 化、記憶、思維、回饋

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```
